# 콘솔프로젝트_PilatesClass

## 주제

필라테스 수업, 회원 관리 프로그램


## 주제선정 이유

MVC 디자인 패턴과 CRUD(생성, 읽기, 갱신, 삭제)를 Java와 MySQL을 이용하여 구현하고
객체지향 프로그래밍의 특징(클래스의 구성 멤버, 상속 등)을 적극 활용하기 위해,
저희 팀은 사용자가 일상생활에서 사용할 수 있는 수업, 회원 관리 프로그램을 주제로 선정하였습니다.
